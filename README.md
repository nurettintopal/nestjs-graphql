# Usage

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Open a browser and go to:
```sh
http://localhost:3000/graphql
```
